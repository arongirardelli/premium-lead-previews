# AGENTS.md — Instruções para Codex

## Contexto

Este repositório é usado para criar prévias estratégicas de landing pages, sites e entregas personalizadas para leads comerciais de Aron Girardelli.

As páginas criadas aqui são demonstrações visuais e estratégicas, não sites oficiais dos leads.

## Objetivo

Criar páginas premium, responsivas, mobile-first e com foco em conversão, autoridade e clareza comercial.

## Regras obrigatórias

- Não inventar dados sobre empresas, produtos, preços, cardápios ou prêmios.
- Usar apenas informações fornecidas no briefing da tarefa.
- Não afirmar que a página é oficial do lead.
- Quando necessário, usar linguagem de prévia estratégica.
- Não criar promessas comerciais impossíveis de verificar.
- Não usar imagens privadas ou não fornecidas.
- Usar placeholders visuais elegantes quando imagens reais não estiverem disponíveis.
- Manter o código limpo, simples e fácil de editar.
- Priorizar performance e responsividade.
- A página deve funcionar muito bem no celular.
- Incluir meta robots noindex,nofollow em páginas demonstrativas.
- Não quebrar páginas existentes.

## Padrão visual

- Visual premium
- Hierarquia forte
- CTAs claros
- Pouco texto por dobra
- Muito respiro
- Layout elegante
- Sem aparência de template genérico

## Antes de finalizar

- Rodar build, lint ou testes disponíveis no projeto.
- Corrigir erros encontrados.
- Informar arquivos alterados.
- Informar comandos executados.
- Informar caminho final da página.
