# LEAD_PREVIEW_CHECKLIST

Utilize este checklist antes de finalizar e apresentar a landing page:

- [ ] Dados reais usados?
- [ ] Nenhum dado inventado?
- [ ] Está claro que é prévia não oficial?
- [ ] Primeira dobra impressiona?
- [ ] Página está bonita no celular?
- [ ] CTA principal é claro?
- [ ] Visual parece premium?
- [ ] Há prova social organizada?
- [ ] Há diagnóstico sem crítica agressiva?
- [ ] Há mockup ou demonstração visual?
- [ ] Há CTA final leve?
- [ ] Build passou?
- [ ] Lint passou?
- [ ] Typecheck passou?
- [ ] Meta robots noindex,nofollow configurada?
