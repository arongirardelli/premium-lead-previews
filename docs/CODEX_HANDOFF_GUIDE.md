# CODEX_HANDOFF_GUIDE

Como transformar um briefing de lead em página:

1. Ler dossiê.
2. Ler Brand Sheet.
3. Ler entrega personalizada.
4. Separar dados reais de recomendações.
5. Criar rota em `/leads/[slug]`.
6. Criar visual premium.
7. Não inventar imagens, números, prêmios, preços ou serviços.
8. Rodar lint, typecheck e build.
9. Revisar mobile.
10. Informar arquivos alterados.
