# PIPELINE_WORKFLOW

Este repositório hospeda prévias estratégicas para leads comerciais.

As páginas não são sites oficiais dos leads.
Objetivo: gerar percepção de valor, curiosidade e abertura de conversa.

## Fluxo completo:
1. Prospecção
2. Dossiê do lead
3. Brand Sheet
4. Entrega personalizada
5. Briefing técnico
6. Landing demonstrativa
7. Revisão visual
8. Publicação
9. Outreach
10. Follow-up

## Diretrizes de Conteúdo
- Não mencionar preço na landing.
- Não inventar dados.
- Não criar cardápio, serviços, números ou promessas sem fonte.
- Sempre manter tom consultivo, premium e respeitoso.
- A página deve demonstrar oportunidade, não atacar o lead.
