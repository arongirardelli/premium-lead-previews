# LANDING_PAGE_VISUAL_PLAYBOOK

## Padrão Visual Obrigatório

- As páginas devem parecer entregas premium, não wireframes.
- Visual 70% atmosfera / 30% texto.
- Mobile-first.
- Primeira dobra precisa gerar impacto imediato.
- Usar profundidade, textura, mockups, selos, cards, divisórias, fundos, shapes e microinterações sutis.
- Evitar páginas muito textuais.
- Evitar aparência genérica.
- Não depender de imagens inventadas.
- Se não houver imagens reais, criar composição visual com CSS, textura, formas, mockups e elementos editoriais.

## Hierarquia Forte

- **Headline**: Curta e forte.
- **Subtítulo**: Claro.
- **CTA**: Visível.
- **Prova Social**: Destacada.
- **Diagnóstico Consultivo**.
- **Mockup** do que poderia melhorar.
- **CTA Final** de baixo atrito.

**Toda landing deve parecer boa o suficiente para ser enviada ao lead como prévia profissional.**

## Regras Específicas por Nicho

- **Restaurantes / Delivery**: Foco no visual dos pratos (se houver fotos reais), ambientação e experiência, facilidade de pedir.
- **Clínicas / Profissionais Premium**: Passar autoridade, usar tons sóbrios, fotos da estrutura e equipe (se fornecidas), prova social sólida.
- **Salões / Beleza**: Minimalismo e elegância, foco na transformação e bem-estar, agenda fácil.
- **Lojas Locais**: Foco em conveniência, destaque para produtos principais ou localização, layout amigável e conversivo.
