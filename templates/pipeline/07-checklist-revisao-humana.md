# 07 · Checklist — Revisão Humana

Antes de enviar qualquer mensagem, proposta ou entrega, revisar:

## 1. Dados do lead

- [ ] Nome da empresa está correto
- [ ] Nome do responsável está correto, se usado
- [ ] Nicho está correto
- [ ] Cidade/bairro está correto
- [ ] Site/Instagram/Google Maps foram conferidos
- [ ] Contato é público
- [ ] Nenhum dado foi inventado

## 2. Personalização

- [ ] A mensagem cita algo real do lead
- [ ] A entrega parece feita para aquele negócio
- [ ] Não parece template genérico
- [ ] A dor identificada é verdadeira
- [ ] A oportunidade está clara

## 3. Tom

- [ ] Está consultivo
- [ ] Não está agressivo
- [ ] Não critica o lead de forma ofensiva
- [ ] Não parece spam
- [ ] Não promete resultado impossível
- [ ] Não usa urgência falsa

## 4. Oferta

- [ ] O primeiro contato não menciona preço
- [ ] A proposta só aparece após interesse
- [ ] O valor vem depois do diagnóstico e da solução
- [ ] Os pacotes fazem sentido para o porte do lead
- [ ] O escopo está claro

## 5. Entrega visual

- [ ] Está legível no celular
- [ ] Usa identidade visual coerente
- [ ] Tem CTA claro
- [ ] Mostra valor rápido
- [ ] Não está poluída
- [ ] Não exagera nos textos

## 6. Envio

- [ ] Canal correto escolhido
- [ ] Link da entrega funciona
- [ ] Link da proposta funciona, se houver
- [ ] Rascunho salvo no Gmail, se for e-mail
- [ ] CRM atualizado
- [ ] Próximo follow-up definido
